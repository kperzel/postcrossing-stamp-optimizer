# 📬 USPS Stamp Optimizer for Postcrossing

A Shiny app that helps you find the optimal combination of stamps 
you already own to cover current international USPS postage rates.

## Features
- Displays current USPS domestic and international postage rates
- Select which stamp denominations you own
- Add custom/legacy stamp values
- Calculates top 10 optimal combinations for international letter postage
